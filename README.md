# Learn Music Theory
A CLI music theory training app that drills: 
* Major Key accidentals
* Chord Tones

## Features
* Randomzied key testing (no repeats until all keys are used) 
* Accidentals quiz (number first, then identify sharps/flats)
* Chord tone drills (Major, Minor, Dominant 7)
* Immediate feedback after every input

## What it helps with 
* Memorizing the circle of fifths
* Learning key signatures
* Understanding chord construction
* Quickly identifying chord tones when improvising

## How to run it
Download the specified files and run with python3 in CLI
